## 💫 About Me

Sono un **Junior Web Developer** dedicato alla creazione di interfacce web pixel-perfect, responsive e user-friendly. Ho una forte passione per il **UI/UX design** e preferisco il controllo totale del codice personalizzato (**Flexbox & Grid**) per dare vita a esperienze digitali uniche e sartoriali, senza dipendere esclusivamente da framework pre-costruiti. 

Mi entusiasma trasformare problemi complessi in codice pulito ed efficiente, spingendo costantemente i miei limiti sia nel frontend che nell'ecosistema **MERN stack**.

* 🔭 **Attualmente sto lavorando su:** Progetti full-stack complessi e affinamento delle performance frontend.
* 🌱 **Sto approfondendo:** TypeScript avanzato e metodologie di UI avanzate.
* ⚡ **Fun Fact:** Affronto il refactoring del codice con la stessa grinta con cui scendo in campo.
